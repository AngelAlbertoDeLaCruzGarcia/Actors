### Pasos para ejecutar el proyecto
 

* Clonar el proyecto
```
git clone https://github.com/AngelAlbertoDeLaCruzGarcia/Actores.git
```
* Instalar los paquetes
```
npm install
```
* Ejecutar el proyecto
```
expo start
```
